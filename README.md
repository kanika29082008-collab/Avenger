# Avenger Hand Tracking Demo

A browser-based hand tracking demo using MediaPipe Hands, HTML, CSS, and JavaScript.
This project recognizes simple hand gestures and displays Avengers-inspired visual effects:

- **Open palm** → Iron Man repulsor effect
- **Spider web sign** → Spider-Man web effect
- **Fist** → Wanda energy orb effect

## Run locally

1. Install Python if you do not already have it.
2. Run:

```bash
npm start
```

3. Open `http://localhost:5500` in your browser.

## Files

- `index.html` — main page and UI
- `styles.css` — project styling
- `script.js` — MediaPipe hand tracking and gesture effects

## Notes

This is a static browser demo. Use a local server to avoid browser camera restrictions.
